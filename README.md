## 🌟 Overview
A highly scalable architectural starting point for enterprise backends. This NestJS boilerplate comes pre-configured with Redis transport layers, strict TypeScript linting, and modular structures ideal for microservice communication.

## ✨ Key Features
- 🔌 **Redis Transport**: Pre-configured for pub/sub microservice messaging.
- 🏗️ **Modular Design**: Domain-driven directory structure.
- 🛡️ **Strict Typings**: Advanced TypeScript configurations for safe coding.

## 🚀 Getting Started

### Prerequisites
Make sure you have `TypeScript_NestJS` installed on your local machine.

### Installation
```bash
git clone https://github.com/HafezMosleh/nestjs-microservice-base.git
cd nestjs-microservice-base
npm install
```

### Usage
```bash
npm run start:dev
```
