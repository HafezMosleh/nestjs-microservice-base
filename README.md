# nestjs-microservice-base

NestJS boilerplate for microservices with RabbitMQ and Redis caching.

## Features
- 🚀 Fast and optimized
- 🔒 Secure by default
- 🛠 Easy to configure

## Installation
```bash
git clone https://github.com/HafezMosleh/nestjs-microservice-base.git
cd nestjs-microservice-base
```
